== Installation ==

1. Upload `wp-group-access.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Change settings in admin panel

== Screenshots ==

1. Add group in plugin menu.
2. Go to menu Users and start editing user. Change group for user.
3. Go to Category editing.  And select groups for read and post messages.
